CSS3 introduced many features that significantly enhanced web design capabilities, including improved styling, layout techniques, and responsiveness. Here are the core features of CSS3:

1. Selectors
CSS3 introduced advanced selectors, allowing for more precise styling:
Attribute selectors ([type="text"])
Pseudo-classes (:nth-child(), :not(), :empty, etc.)
Pseudo-elements (::before, ::after, etc.)
2. Box Model Enhancements
Border-radius: For rounded corners (border-radius).
Box-shadow: Adds shadow effects to elements.
Outline-offset: Adjusts the space between an element and its outline.
3. Backgrounds
Multiple backgrounds: Apply multiple background images to a single element.
Gradient backgrounds: Linear and radial gradients without using images.
Background-size: Control the size of background images.
Background-clip: Defines the area of the element where the background applies.
4. Text Effects
Text-shadow: Add shadows to text.
Word-wrap/overflow-wrap: Manage long words or content in text blocks.
Custom web fonts: Use @font-face to load and use custom fonts.
5. Flexbox and Grid Layouts
Flexbox: Enables easy alignment, spacing, and layout of elements in one-dimensional layouts.
CSS Grid: Provides powerful two-dimensional layout capabilities.
6. Transitions and Animations
Transitions: Smooth animations between property changes (transition).
Animations: Define keyframe animations (@keyframes).
7. Transformations
2D Transforms: Scale, rotate, translate, or skew elements.
3D Transforms: Add depth with 3D effects (transform: rotateX(), rotateY()).
8. Media Queries
Enable responsive design by applying different styles based on screen size, orientation, resolution, etc.
9. New Color Models
RGBA and HSLA color formats for transparency and hue/saturation control.
10. Responsive Units
New units for better control: vw, vh, vmin, vmax (viewport-relative units).
11. Box-sizing
Control how the width and height of an element are calculated (content-box or border-box).
12. Columns and Multi-column Layout
Divide content into multiple columns using properties like column-count, column-gap, and column-rule.
13. Clipping and Masking
Clip or mask elements using shapes and images (clip-path, mask).
14. New Pseudo-classes
:nth-child(), :first-of-type, :last-of-type, :only-child, and more.
15. Custom Properties (CSS Variables)
Define reusable variables using -- syntax, e.g., --main-color: #333;.
16. Improved Form Styles
Styling for form elements like :required, :valid, :invalid.
These features combined allow designers to create visually appealing, modern, and responsive websites with greater ease and flexibility.
